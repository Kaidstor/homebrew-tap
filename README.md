# Kaidstor's Homebrew tap

```bash
brew install kaidstor/tap/sql-kai
```

| Cask | Description |
| --- | --- |
| [`sql-kai`](Casks/sql-kai.rb) | [Desktop Postgres client](https://github.com/Kaidstor/sql-kai) with SSH tunnels, encrypted vault and a CLI twin for AI agents — signed and notarized |
| [`sec`](Casks/sec.rb) | [Local secrets manager](https://github.com/Kaidstor/sec) CLI (managed by GoReleaser) |
| [`tcp-client`](Casks/tcp-client) | Legacy TCP client on Tauri |

The GUI apps keep themselves up to date via their built-in updater.
